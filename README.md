## âœ¨ Key Features

- **Civic Feed**: AI-curated video feed of parliamentary sessions, county assembly debates, and civic town halls.
- **Hero Player**: Live streaming of major events (Citizen TV Live, Parliament Live) with "Cinematic Mode" for immersive viewing.
- **Truth Layer**: Automated fact-checking and context overlays on political speeches and claims.

- **Interactive Simulation**: "Budget Mastery" and policy simulators to understand complex governance trade-offs.

## ğŸ Getting Started

1.  **Clone the repository**

    ```bash
    git clone https://github.com/sammy001-svg/polify-kenya.git
    cd polify-kenya
    ```

2.  **Install dependencies**

    ```bash
    npm install
    ```

3.  **Environment Setup**
    Create a `.env.local` file with your Supabase credentials:

    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_project_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
    ```

4.  **Run the development server**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
